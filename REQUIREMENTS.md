# Install Commands

Here are the one-line commands to install all necessary modules for the Authentication App.

## Server

Run this command inside the `server` directory:

```bash
npm install bcryptjs cookie-parser cors dotenv express express-rate-limit express-validator helmet jsonwebtoken mongoose nodemailer nodemon
```

## Client

Run these commands inside the `client` directory:

### Core Dependencies

```bash
npm install axios lucide-react react react-dom react-hot-toast react-router-dom
```

### Development Dependencies

```bash
npm install -D autoprefixer eslint eslint-plugin-react eslint-plugin-react-hooks eslint-plugin-react-refresh postcss tailwindcss vite @types/react @types/react-dom @vitejs/plugin-react
```
